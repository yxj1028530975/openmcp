# OpenMCP 应用模块

本目录包含 OpenMCP 平台的所有应用模块，每个子目录是一个独立的应用服务。

## 应用列表

### 1. 天气服务 (weather)

提供基于城市名称或经纬度获取天气信息的API服务。

- 通过城市名称获取天气信息
- 通过经纬度获取天气信息

[详细文档](weather/README.md)

### 2. 今日热榜 (DailyHotApi)

提供各大平台热榜数据的聚合服务，支持50多个平台。

- 获取知乎、微博、B站等热榜数据
- 支持聚合新闻热榜查询
- 提供全面的热门平台数据

[详细文档](DailyHotApi/README.md)

## 应用结构

每个应用模块通常包含以下文件：

- `xxx_api.py`: 应用的主要API实现
- `README.md`: 应用的详细说明文档
- 其他辅助文件: 如工具函数、数据模型等

## MCP 集成

所有应用都已集成到 MCP（Multi-Channel Protocol）系统中，通过统一的 `/mcp` 接口可以调用任何应用的功能。

调用格式：

```json
{
  "operation": "{app-name}.{operation_id}",
  "parameters": {
    // 操作需要的参数
  }
}
```

## 开发新应用

要开发新的应用模块，请参考现有模块的结构，确保：

1. 创建一个新的子目录
2. 实现一个FastAPI应用对象
3. 提供详细的README文档
4. 在主应用中注册该模块

详细开发指南请参考项目根目录的文档。
